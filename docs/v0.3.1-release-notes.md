# v0.3.1 Release Notes

v0.3.1 is the latest stable release of Todoist Assistant.

## What this release covers
- Release metadata updated to `v0.3.1` across the Python package, frontend package metadata, and release docs.
- README links now point at the `todoist-assistant-v0.3.1` GitHub release tag.
- The docs index now points at the `v0.3.1` release notes page.

## Release checklist
- Confirm `make typecheck`, `make lint`, `make test`, and `make coverage` are green.
- Publish the `todoist-assistant-v0.3.1` release tag and assets.
- Keep the stable docs, README, and package metadata aligned with `0.3.1`.
